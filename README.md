# Gerenciador de Tarefas com Flask

Projeto CRUD de gerenciamento de tarefas desenvolvido com Flask.

## Tecnologias

- Flask
- SQLAlchemy
- SQLite
- Docker
- Docker Compose

## Como executar

Clone o repositório:

git clone https://github.com/seuusuario/gerenciador-tarefas-flask.git

Entre na pasta:

cd gerenciador-tarefas-flask

Execute com Docker:

docker compose up

A aplicação estará disponível em:

http://localhost:5000
